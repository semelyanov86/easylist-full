# Easylist

Персональный ассистент для управления поиском работы и повседневными списками покупок. Отслеживание откликов на вакансии, AI-анализ компаний, генерация сопроводительных писем, Kanban-доска статусов и многое другое.

## Стек

| Слой          | Технологии                                                 |
|---------------|------------------------------------------------------------|
| Backend       | PHP 8.4, Laravel 12, Laravel Fortify, Laravel Sanctum      |
| Frontend      | Vue 3, TypeScript, Inertia.js v2, Tailwind CSS v4          |
| Сборка        | Vite 7, SSR                                                |
| Маршрутизация | Laravel Wayfinder (type-safe TS-функции для роутов)        |
| Качество кода | PHPStan (max), ESLint + FSD-плагин, Prettier, Pint, Rector |
| Тесты         | PHPUnit 13                                                 |
| Production    | Laravel Octane (Swoole), Supervisor, systemd               |

## Основные возможности

### Трекер вакансий

- Добавление вакансий с данными: название, компания, город, зарплата, ссылка на вакансию, ссылка на резюме
- **Два режима просмотра**: таблица с пагинацией и Kanban-доска с перетаскиванием между статусами
- **Пользовательские статусы** с настраиваемыми цветами (15 цветов) и порядком
- **Категории вакансий** с привязкой валюты (RUB/USD/EUR)
- **Избранные вакансии** — быстрый доступ к важным откликам
- Фильтрация по статусу, категории, дате и поиск по названию/компании
- Soft delete — удалённые вакансии можно восстановить

#### Вкладки вакансии

| Вкладка | Описание |
|---------|----------|
| Обзор | Основная информация, теги навыков, лента активности |
| Комментарии | Заметки и комментарии к вакансии |
| Документы | Файлы и ссылки (резюме, портфолио, рекомендации, сертификаты, сопроводительные письма) |
| Компания | AI-анализ компании (отзывы, новости, стек технологий) |
| Контакты | Контактные лица (email, телефон, LinkedIn, Facebook, WhatsApp) |
| Задачи | Задачи по вакансии с дедлайнами и синхронизацией с TickTick |

#### Публичный доступ

Вакансию можно расшарить по UUID-ссылке — получатель увидит основную информацию без авторизации.

### AI-функции

| Анализ компании | Отзывы, новости, стек, ключевые ссылки | — |
| Извлечение навыков | Автоматическое определение тегов из описания вакансии | — |
| Поиск контактов | Поиск рекрутеров и HR по LinkedIn/email | — |

| Сопроводительное письмо | Генерация персонализированного cover letter | Premium |

### Списки покупок

- Папки для организации списков (с иконками)
- Списки покупок с товарами: название, описание, количество, тип, цена
- Отметка товара как купленного, звёздочка для приоритетных
- Перетаскивание для изменения порядка (списки, папки, товары)
- Публичный доступ по ссылке (UUID)
- Отправка списка по email
- Массовые действия: снять все отметки, удалить все товары

### Дашборд

7 виджетов с ленивой загрузкой и skeleton-анимацией:

- **Недавняя активность** — лента изменений по вакансиям
- **Задачи** — невыполненные задачи с дедлайнами
- **Воронка вакансий** — распределение по статусам (с фильтром по категории)
- **Избранные вакансии** — закреплённые вакансии
- **Последние вакансии** — недавно добавленные
- **Востребованные навыки** — топ навыков по всем вакансиям
- **Динамика откликов** — график подачи заявок во времени

### Аутентификация

- Регистрация и вход по email/паролю
- Сброс и смена пароля
- Подтверждение email
- **Двухфакторная аутентификация (2FA)**: TOTP (Google Authenticator, Authy и др.)
- **WebAuthn / Passkeys**: вход по биометрии или аппаратному ключу (FIDO2)
- Коды восстановления
- **API-токены** (Laravel Sanctum) для внешних интеграций

### Настройки

- Профиль: имя, email, раздел «О себе»
- Смена пароля
- Внешний вид: светлая / тёмная тема / системная
- Управление 2FA и WebAuthn-ключами
- API-токены: создание и отзыв
- Пользовательские статусы вакансий
- Интеграция с TickTick (токен и ID списка)

### PWA

Приложение поддерживает установку на устройство как Progressive Web App (standalone-режим, иконки, тёмная тема).

### REST API (v1)

Полноценный API с аутентификацией через Sanctum:

- `/api/v1/me` — текущий пользователь
- `/api/v1/jobs` — CRUD вакансий, смена статуса, избранное, шаринг
- `/api/v1/jobs/{job}/comments` — комментарии
- `/api/v1/jobs/{job}/documents` — документы
- `/api/v1/jobs/{job}/contacts` — контакты
- `/api/v1/job-categories` — категории и вакансии по категории
- `/api/v1/statistics` — статистика для дашборда
- `/api/v1/tasks/pending` — незавершённые задачи
- `/api/v1/lists`, `/api/v1/items`, `/api/v1/folders` — списки покупок
- Публичные эндпоинты: `/api/v1/public/jobs/{uuid}`, `/api/v1/links/{uuid}`

## Архитектура

**Backend** — Domain-Driven Design (DDD): тонкие контроллеры, бизнес-логика в отдельных классах, Eloquent-модели только для инфраструктуры.

**Frontend** — Feature-Sliced Design (FSD):

```
resources/js/
├── app/           # Инициализация приложения
├── pages/         # Страницы (Inertia)
├── widgets/       # Сложные UI-блоки (AppShell, Sidebar, Header, дашборд-виджеты)
├── features/      # Пользовательские сценарии (auth, job, shopping, settings)
├── entities/      # Бизнес-сущности (user, job, contact, skill и др.)
└── shared/        # UI-компоненты, composables, утилиты, типы
```

Импорты строго по иерархии: `pages` → `widgets` → `features` → `entities` → `shared`.

## Установка

```bash
git clone <repo-url> easylist
cd easylist

# Установка зависимостей
composer install
npm install

# Конфигурация
cp .env.example .env
php artisan key:generate

# База данных
php artisan migrate --seed

# Wayfinder (генерация TS-функций для роутов)
php artisan wayfinder:generate

# Сборка фронтенда
npm run build
```

Или через Task-runner:

```bash
task install
```

## Разработка

```bash
# Запуск dev-сервера (Vite + Laravel)
composer run dev

# Или по отдельности
php artisan serve
npm run dev
```

## Управление пользователями

```bash
# Создать пользователя (пароль будет сгенерирован и показан в консоли)
php artisan user:create "Иван Иванов" ivan@example.com

# Создать пользователя с premium-статусом
php artisan user:create "Иван Иванов" ivan@example.com --premium

# Переключить premium-статус (включить/выключить)
php artisan user:toggle-premium ivan@example.com

# Удалить пользователя (с подтверждением, удаляет вакансии и документы)
php artisan user:delete ivan@example.com

# Удалить без подтверждения
php artisan user:delete ivan@example.com --force

# Синхронизация задач с TickTick
php artisan ticktick:sync
```
